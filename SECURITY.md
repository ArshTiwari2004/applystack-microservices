# Security Policy

## Supported Versions

The following versions of this project are currently supported with security updates:

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |
| < 1.0   | No        |

## Reporting a Vulnerability

If you discover a security vulnerability in ApplyStack, please report it responsibly.

- Open an issue in this repository with detailed information about the vulnerability  
- Include steps to reproduce, potential impact, and possible fixes if available  

I will reply within 48 hours. will review valid reports, and fixes will be prioritized accordingly.

Please avoid publicly disclosing the vulnerability until it has been resolved.
